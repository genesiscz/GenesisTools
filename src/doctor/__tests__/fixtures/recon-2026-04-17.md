## Simulators (real shape)
- `xcrun simctl list devices --json` returns `{ devices: { "<runtime-id>": [{ udid, name, deviceTypeIdentifier, state, dataPath, dataPathSize, logPath, isAvailable }, ...] } }`
- **`dataPathSize` is data dir only (~18 MB/sim)** — real footprint needs `du -sk <dataPath>` AND the runtime bundle itself (GBs).
- `device.plist` contains: `deviceType`, `isDeleted`, `isEphemeral`, `name`, `runtime`, `runtimePolicy`, `state`, `UDID`. **No `lastBootedAt` or last-use timestamp.**
- **Workaround**: use `stat -f %m <logPath>` or walk `~/Library/Logs/CoreSimulator/<UDID>/` for newest file mtime → infer last use.
- Real runtimes installed: `iOS 26.3`, `iOS 26.4`. Value of "delete runtime for unused iOS version" ≫ value of "delete individual sims".
